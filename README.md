# Прирост средней конверсии по магазинам Санкт-Петербурга

Тестовое задание: расчёт прироста средней конверсии на 1 магазин в Санкт-Петербурге между двумя периодами.

## Задача

По данным о трафике и покупателях рассчитать прирост средней конверсии для LFL-магазинов Санкт-Петербурга (магазинов, работавших в обоих периодах).

## Логика решения

1. **Конверсия** — отношение покупателей к входящим: `Покупатели / Входящие`. Строки с нулевым трафиком исключаются.
2. **LFL-магазины** — определяются через `СЧЁТЕСЛИМН`: магазин считается сопоставимым, если его код присутствует и в периоде 1, и в периоде 2.
3. **Средняя конверсия** по каждому периоду считается через `СРЗНАЧЕСЛИМН` с фильтрами по городу (Санкт-Петербург), LFL и номеру периода.
4. **Прирост** — `(Период 2 − Период 1) / Период 1`.

## Результат

| Показатель | Значение |
|---|---|
| Средняя конверсия, период 1 | 12,49% |
| Средняя конверсия, период 2 | 11,30% |
| **Прирост** | **−9,48%** |

Средняя конверсия по LFL-магазинам Санкт-Петербурга снизилась на 9,48% во втором периоде по сравнению с первым.
